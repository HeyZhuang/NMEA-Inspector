**1.3** ***\*基础视图\****

![img](file:///C:\Users\LENOVO\AppData\Local\Temp\ksohtml3400\wps4.jpg) 

**基础视图示意图**

基础视图显示最常用的时空信息，包括纬度、经度、海拔、时间、日期，具体格式要求如下：

| ***\*字段\**** | ***\*格式要求\****                                           |
| -------------- | ------------------------------------------------------------ |
| ***\*纬度\**** | 将原始的度分格式转换成度格式，携带单位”°”；北纬用正数表示，南纬用负数表示；小数点后保留6位小数；格式：WW.wwwwww |
| ***\*经度\**** | 将原始的度分格式转换成度格式，携带单位”°”；东经用正数表示，西经用负数表示；小数点后保留6位小数；格式：JJJ.jjjjjj |
| ***\*海拔\**** | 携带单位”m”；小数点后保留2位小数；格式：AA.aa                |
| ***\*时间\**** | UTC时间；格式：hh:mm:ss（hh为时，mm 为分，ss为秒）           |
| ***\*日期\**** | UTC日期；格式：yyyy:MM:dd（yyyy为年，MM为月，dd为日）        |

可拓展一些自定义的数据项，比如“北京时间”等等；
